파이어베이스를 이용한 구글 로그인 흐름 요약

* 파이어베이스 사이트에서 Authenticate 셋팅하고,
웹 프로젝트 생성하고, 개인정보 받기

* React 에서 .env 에 파이어베이스 개인정보 적기
(보안을 아주 철저히 하고 싶으면, 이 정보도 서버에 저장하고, 서버에서 받는식으로 해야되요)

* firebaseConfig.ts, GoogleSignInCompo.tsx 이런 구글에서 적어놓은 코드 뜯어와서
붙여넣기

* LoginPage.tsx 같이 내가 로그인 구현할 페이지 만들기

* onSignInSuccess, onSignInError 에 넘겨줄 함수 만들고, 컴퍼넌트 호출할때
두 함수 인자로 넘겨주기

* handleSuccess (onSignInSuccess 에 넘겨준 함수) 에 내가 하고싶은 기능 넣기

* 서버에서 준 token(유저의 개인정보)을 store 에 저장하기
